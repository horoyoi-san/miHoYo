启动教程：
1.双击BocchiSR.exe，确保窗口启动。
2.进入\Proxy文件夹，双击启动VariousArtists.Proxy.exe。（如果窗口一闪而过，请安装Net.9）
3.进入游戏，账号密码随便输入。
4.更换面板请在https://srtools.pages.dev/设置面板，并放进\settings文件夹的同名文件替换。
5.享受游戏。

Start the tutorial:
1. Double-click BocchiSR.exe to make sure the window starts.
2. Go to the \Proxy folder, double click to start VariousArtists.Proxy.exe. (If the window flickers, install Net.9)
3. Enter the game and enter your account password at random.
4.To replace the panel, please find it at https://srtools.pages.dev/设置面板 and put the same name file in the \settings folder to replace it.
5. Enjoy the game.
